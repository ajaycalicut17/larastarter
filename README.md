# larastarter
Laravel starter

## Installation

You can install the package via composer:

``` bash
composer require ajaycalicut17/larastarter --dev
```

Run artisan command for installation:

``` bash
php artisan larastarter:install
```

Run npm

``` bash
npm install && npm run dev
```

Database migration

``` bash
php artisan migrate
```
